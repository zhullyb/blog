Una aplicación web tradicional apilada consiste en:

  • un sistema operativo (Linux)
  • un servidor web, como Apache o nginx
  • una DBMS relacional, como MySQL o PostgreSQL
  • un lenguaje de programación, como PHP, Python o Perl

Un ejemplo de tal apilamiento es LAMP.
