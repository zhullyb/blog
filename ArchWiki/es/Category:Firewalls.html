Arch Linux viene con dos opciones para administrar un cortafuegos, ninguno de las cuales se habilita automáticamente. El kernel de Linux incluye la estructura de filtrado de paquetes netfilter que puede administrarse por cualquiera de los siguientes:

  • iptables es la utilidad de espacio de usuario tradicional para administrar un cortafuegos. La configuración se puede realizar directamente a través de las utilidades del espacio de usuario o instalando una de las diversas herramientas de configuración de la GUI.
  • nftables es un proyecto más nuevo que reemplaza la estructura iptables. Combina una sintaxis simple con similares características y beneficios de rendimiento sobre iptables.
