Nessus es un escáner de vulnerabilidades patentado, disponible de forma gratuita para uso personal. Hay más de 40,000 complementos que cubren una amplia gama de fallos tanto locales como remotos.

Instalación

Instale el paquete nessusAUR.

Configuración posterior a la instalación

Registre su correo electrónico en [1] y espere a que se le envíe su clave por correo electrónico.

Utilización

El paquete nessusAUR proporciona un archivo de unidad nessusd.service, véase systemd (Español)#Utilizar las unidades para más detalles.

Acceda a la interfaz web en https://localhost:8834 y/o utilice la interfaz de línea de comandos (/opt/nessus/sbin/nessuscli). En la mayoría de los navegadores, deberá aceptar manualmente el certificado SSL que creó para el servidor.

Licencia

Detenga el demonio nessus antes de hacer algo con /nessuscli. 

# systemctl stop nessusd.service

Active la licencia: 

# nessuscli fetch --register <Código de activación>

Consulte su código de activación de licencia actual: 

# nessuscli fetch --code-in-use

Actualización de complementos

Detenga el demonio nessus antes de hacer algo con /nessuscli. 

# systemctl stop nessusd.service

Actualize los complementos: 

# nessuscli update --plugins-only

Eliminación

El paquete puede eliminarse con pacman, pero los archivos creados por Nessus, como la base de datos de complementos que descarga, deben eliminarse manualmente:

Nota: Esto eliminará sus archivos de configuración de Nessus.
# rm -r /opt/nessus
