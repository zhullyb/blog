Nestopia es un emulador de la NES de código abierto que intenta emular su hardware con la mayor precisión posible. Está disponible para Windows, macOS, Linux y BSD. También hay un port para Libretro, véase RetroArch para obtener más información.

Instalación

Instale el paquete nestopiaAUR del AUR. Alternativamente, instale nestopia-gitAUR para la versión de desarrollo.

Solución de problemas

No hay audio con el backend libao

Actualmente, el backend libao está roto. Se recomienda cambiar la API de audio a SDL en Emulador -> Configuración -> Audio.
