#archlinux-wiki é o canal IRC na rede Freenode para o ArchWiki. Esse canal é fornecido como um meio mais rápido de discutir as edições do wiki, bem como fazer perguntas gerais sobre o ArchWiki e sobre ele.

Uso

Entrar nos canais requer um cliente IRC; veja IRC para detalhes. #archlinux-wiki tem o modo de canal definido para +r, o que significa que é necessário se identificar com NickServ para entrar.

Regras do #archlinux-wiki

O tópico central para o canal é perguntas a cerca do ArchWiki e seus artigos. Perguntas de suporte em geral devem ser perguntadas no #archlinux ou nos fóruns do Arch.

  • Sempre siga as três regras fundamentais
    Discussões ao vivo não são um substituto para resumos de edição adequados, ou páginas de discussão de artigo. Sempre siga ArchWiki:Contribuindo#As 3 regras fundamentais.
    Logs do canal podem ser publicados em páginas de discussão de artigo, ou resumido para referência posterior.
  • Mantenha-se educado e no tópico
    Siga o Código de conduta e as regras do #archlinux.
  • Operadores
    ArchWiki:Administradores com uma conta IRC verificada têm direitos de operador e são indicados como tal neste canal (@nick). Administradores podem apontar outros operadores para aplicar o Código de conduta.

Supybot

O bot do canal, phrik, usa Limnoria, que é uma versão modificada do supybot.

Faça link para um artigo wiki no canal usando o DuckDuckGo: 

!daw artigo

Faça link para um relatório de erro: 

!bug número_da_tarefa
