Utilità

Gestori di pacchetti

Si veda anche: AUR helpers and Pacman GUI Frontends.

  • Pacmatic — Un wrapper di pacman che controlla le news di Arch Linux prima di procedere ad un aggiornamento, evita gli aggiornamenti parziali e avvisa nel caso di cambiamenti nei file di configurazione.
http://kmkeen.com/pacmatic || pacmatic
  • pkgfile — Un tool che cerca a quale pacchetto corrisponde un file.
https://github.com/falconindy/pkgfile || pkgfile
  • pkgtools — Una collezione di script per creare pacchetti i Arch Linux.
https://github.com/Daenyth/pkgtools || pkgtoolsAUR
  • srcpac — Un semplice tool che automatizza la ricompilazione di un pacchetto dai sorgenti.
https://projects.archlinux.org/srcpac.git || srcpac[broken link: package not found]

Programmi per visualizzare lo spazio su disco

  • Disk Usage Analyzer (Baobab) — Un analizzatore dello spazio su disco per il desktop GNOME.
http://www.marzocca.net/linux/baobab || gnome-utils[broken link: package not found]
  • Filelight — Un analizzatore dello spazio su disco che crea una mappa interattiva concentrica, i segmenti degli anelli aiutano a visualizzare lo spazio su disco occupato sul proprio computer.
http://methylblue.com/filelight/ || kdeutils-filelight[broken link: package not found]
  • GdMap — Un analizzatore dello spazio su disco che disegna una mappa di rettangoli grandi grandi quanto occupano di spazio i file o le cartelle.
http://gdmap.sourceforge.net/ || gdmap
  • gt5 — Un diff-capable "du-browser".
http://gt5.sourceforge.net || gt5AUR
  • ncdu — Un semplice analizzatore dello spazio su disco in ncurses.
http://dev.yorhel.nl/ncdu || ncdu

Gestore degli appunti

  • Anamnesis — Un manager degli appunti che salva tutto lo storico degli appunti e offre un'interfaccia per effettuare ricerche di testo. Avviabile sia in modalità cli che con interfaccia grafica.
http://anamnesis.sourceforge.net/ || anamnesisAUR
  • ClipIt — Un fork di Parcellite con caratteristiche addizionale e bugfix.
https://sourceforge.net/projects/gtkclipit/ || clipitAUR
  • CopyQ — Un manager degli appunti con ricerca e modifica dello storico, azioni personalizzabili con oggetti e supporto alla riga di comando.
https://github.com/hluk/CopyQ || copyq
  • Glipper — Un gestore degli appunti per il desktop GNOME con molte caratteristiche e supporto a plugin.
https://launchpad.net/glipper || glipperAUR
  • Klipper — Un manager degli appunti completo per il desktop KDE.
https://userbase.kde.org/Klipper || kdebase-workspaceAUR
  • loliclip — Un manager degli appunti sviluppato per gli utenti che usano un WM.
https://github.com/Cloudef/PKGBUILDS/tree/master/loliclip[link interrotto 2020-08-02] || loliclipAUR[broken link: package not found]
  • Parcellite — Un manager degli appunti leggero ma ricco di caratteristiche.
http://parcellite.sourceforge.net/ || parcellite

Tool di compressione

See also Wikipedia:Comparison of file archivers.

Console

  • atool — Uno script per gestire gli archivi in vari formati.
http://www.nongnu.org/atool/ || atool
  • p7zip — Un porting di 7-Zip per i sistemi POSIX, incluso Linux. Il tool da riga di comando è chiamato 7z.
http://p7zip.sourceforge.net/ || p7zip

Interfaccia grafica

  • Ark — Un tool per gestire archivi incluso nel desktop KDE.
https://kde.org/applications/utilities/ark/ || kdeutils-ark[broken link: package not found]
  • File Roller — Un tool per gestire archivi incluso nel desktop GNOME.
http://fileroller.sourceforge.net/ || file-roller
  • p7zip — Un porting di 7-Zip per i sistemi POSIX, incluso Linux. La GUI richiede wxGTK ed è chiamata 7zFM.
http://p7zip.sourceforge.net/ || p7zip
  • PeaZip — Un gestore degli archivi open source.
http://www.peazip.org/peazip-linux.html || peazip-gtk2AUR[broken link: package not found] peazip-qtAUR[broken link: package not found]
  • Squeeze — Un leggerissimo frontend per tool di archiviazione da riga di comando.
http://squeeze.xfce.org/ || squeeze-gitAUR[broken link: package not found]
  • Xarchive — Un gnerico front-end in GTK+ che utilizza dei wrapper esterni per interfacciarsi ai tool di archiviazione da riga di comando.
http://xarchive.sourceforge.net/ || xarchiveAUR[broken link: package not found]
  • Xarchiver — Un leggero gestore degli archivi indipendente dai desktop e scritto in GTK+.
http://xarchiver.sourceforge.net/ || xarchiver

File manager

See also Wikipedia:Comparison of file managers.

Console

  • Dired — Un editor di cartelle integrato con Emacs.
https://www.gnu.org/software/emacs/manual/html_node/emacs/Dired.html || emacs
  • Midnight Commander — Un file manager da console con doppio pannello.
http://www.midnight-commander.org || mc
  • nffm — "Nothing Fancy File Manager", un file manager senza l'utilizzo del mouse scritto in C.
https://github.com/mariostg/nffm || nffm-gitAUR
  • Pilot — Un file manager che è come il client email Alpine.
http://re-alpine.sourceforge.net/ || re-alpineAUR[broken link: package not found]
  • Ranger — Un file manager da console con tasti rapidi alla "vi", personalizzabili, è include alcune caratteristiche.
http://nongnu.org/ranger || ranger
  • Vifm — Un file manager in ncurses con tasti di scelta rapida alla "vi" e visuale a doppio pannello.
http://vifm.sourceforge.net/ || vifm

Interfaccia grafica

  • Andromeda — Un file manager scritto in Qt-based e multi-piattaforma.
https://gitorious.org/andromeda[link interrotto 2020-08-02] || andromeda-gitAUR[broken link: package not found]
  • Dino — Un semplice e potente file manager scritto in Qt.
http://dfm.sourceforge.net/ || dino-dfmAUR[broken link: package not found]
  • Dolphin — Il file manager incluso in KDE4.
http://dolphin.kde.org/[link interrotto 2020-08-02] || kdebase-dolphin[broken link: package not found]
  • Double Commander — File manager con due pannelli divisi. Ispirato a Total Commander e include nuove caratteristiche e idee.
http://doublecmd.sourceforge.net// || doublecmd-gtk2 doublecmd-qt[broken link: replaced by doublecmd-qt5]
  • emelFM2 — Un file manager che implementa la popolare vista a due pannelli.
http://emelfm2.net/ || emelfm2
  • Konqueror — Un file manager per il desktop KDE.
http://www.konqueror.org/ || konqueror
  • Krusader — Un avanzato file manager a due pannelli (in stile Midnight Commander) per il desktop KDE.
http://www.krusader.org/ || krusader
  • GNOME Files — Un estensibile e pesante file manager file manager utilizzato di default in GNOME con supporto a script personalizzati.
http://projects.gnome.org/nautilus/[link interrotto 2020-08-02] || nautilus
  • PCManFM — Un leggero file manager con numerose caratteristiche, navigazione a schede e opzionalmente può gestire il background del desktop.
http://pcmanfm.sourceforge.net/ || pcmanfm
  • QtFileMan — Un file manager simile a PCManFM per LXDE.
http://gitorious.org/qtfileman[link interrotto 2020-08-02] || qtfileman-gitAUR[broken link: package not found]
  • qtFM — Un piccolo e leggero file manager per i desktop Linux basato sulle Qt.
http://www.qtfm.org/[link interrotto 2020-08-02] || qtfmAUR
  • ROX — Un piccolo e veloce file manager che può opzionalmente gestire il background del desktop e i pannelli.
http://rox.sourceforge.net || rox
  • SpaceFM — Un file manager in GTK+ con schede a multi pannelli.
https://ignorantguru.github.com/spacefm/ || spacefmAUR
  • Sunflower — Un piccolo ed altamente personalizzabile file manager con visuale a due pannelli e supporto a plugins.
http://code.google.com/p/sunflower-fm/ || sunflowerAUR
  • Thunar — Un file manager che può essere eseguito come demone per ottenere un avvio rapido e un caricamento delle directory immediato.
http://thunar.xfce.org/index.html[link interrotto 2020-08-02] || thunar
  • Tux Commander — Un file manager a finestra con due pannelli laterali simile ai popolari Total Commander o Midnight Commander.
http://tuxcmd.sourceforge.net/description.php || tuxcmd
  • Worker — Un file manager veloce, leggero e ricco di funzionalità per il sistema X Window.
http://www.boomerangsworld.de/worker/ || workerAUR
  • Xfe — Un file manager simile all'explorer di Microsoft per X (X File Explorer).
http://roland65.free.fr/xfe/index.php/ || xfeAUR

Tool di unione e comparazione file

See also Wikipedia:Comparison of file comparison tools.

  • Diffuse — un piccolo e semplice tool di unione file scritto in Python.
http://diffuse.sourceforge.net/ || diffuse
  • KDiff3 — Un tool per controllare le differenze e unire file e cartelle per il desktop KDE.
http://kdiff3.sourceforge.net/ || kdiff3
  • Kompare — Un frontend per controllere e unire le differenze tra i file. Supporta svariati formati per i diff e fornisce molte opzioni per personalizzare il livello di informazioni da mostrare.
http://www.caffeinated.me.uk/kompare/ || kdesdk-kompare[broken link: package not found]
  • Meld — Un visualizzatore di differenze e tool di unione per comparare i file, cartelle e controllare le versioni di progetti.
http://meld.sourceforge.net || meld

Vim ed Emacs forniscono la funionalità di unione e controllo differenze tramite vimdiff[broken link: invalid section] e ediff.
