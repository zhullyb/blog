This page will explain how to get the Omnikey Cardman 5321 SmartCard Reader up and running under Arch Linux. This guide may work for other models to, but this has not been tested.

Installation

Install the pcsc-tools, and omnikey_ifdokccidAUR packages.

Start pcscd with pcscd.service.

Test with pcsc_scan

You can test the installation by running pcsc_scan. If everything worked fine, you should get something like that:
