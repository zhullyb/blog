Configuration

Disabling UEFI Secure Boot

To disable Secure Boot, set the supervisor password[dead link 2020-03-28 ⓘ] in the BIOS settings. Then you should be able to boot Arch.

Video

Install the xf86-video-intel package as shown here

Solid State Drive

Enable TRIM by mount flag[broken link: invalid section]

Apply trim via periodic fstrim with systemd[broken link: invalid section]

Switch from default CFQ scheduler to NOOP, for the whole system[broken link: invalid section]

Reduce Swappiness to 1

Touchpad

Configure the touchpad as a ClickPad by editing the touchpad section in /etc/X11/xorg.conf.d/50-synaptics.conf as shown here

Troubleshooting

Video

I experienced a loss of h-sync when swiching TTYs[broken link: invalid section] and solved it as suggested by the wiki, by adding i915.enable_ips=0 as a kernel parameter
