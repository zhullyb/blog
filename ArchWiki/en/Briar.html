Briar is a messaging app designed for activists, journalists, and anyone else who needs a safe, easy and robust way to communicate. Unlike traditional messaging apps, Briar doesn’t rely on a central server - messages are synchronized directly between the users’ devices. If the internet is down, Briar can sync via Bluetooth or Wi-Fi, keeping the information flowing in a crisis. If the internet is up, Briar can sync via the Tor network, protecting users and their relationships from surveillance.

Briar is written in Java. Hence, it is platform independent. Besides the primary developed Android- and iOS-Apps there is a headless implementation.

Installation

The command line client for desktops as well as a python wrapper and a graphical user interface can be installed via AUR.
